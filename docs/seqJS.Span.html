Class: Span

seqJS. Span

new Span(_location1, _location2, complement)

Represent a span between two locations. Either both locations can be passed seperately as seqJS.Locations, or the span can be parsed from a Genbank string. By definition location1 < location 2.
Parameters:
Name Type Argument Default Description
_location1 string | seqJS.Location the first location
_location2 seqJS.Location <optional>
 the second location
complement boolean <optional>

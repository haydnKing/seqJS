Class: Feature

seqJS. Feature

new Feature(_type, _location)

Store information about a feature - type: the feature type -- gene, CDS, etc. - location: feature location -- either a FeatureLocation object or string from which one will be built - qualifiers: dictionary Object of qualifiers to be stored with the feature
Parameters:
Name Type Description
_type string the Feature type (e.g. 'gene', 'CDS', ...)
_location seqJS.FeatureLocation | string The feature's location, either as a string in genbank format (e.g. 1..50) or a seqJS.FeatureLocation object

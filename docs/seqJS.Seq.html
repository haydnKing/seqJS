seqJS. Seq

new Seq(_seq, _alphabet, _features, _topology, _length_unit, _strand_type, _residue_type)

An object representing a biological sequence
Parameters:
Name Type Argument Default Description
_seq string the sequence as a string
_alphabet string the alphabet, one of seqJS.Alphabets
_features Array <optional>
 new Array() an array of seqJS.Feature objects referring to the sequence
_topology string <optional>
 'linear' sequence topology, 'linear' or 'circular'
_length_unit string <optional>
 taken from alphabet the length unit of the sequence, one of 'bp', 'aa' or 'rc'
_strand_type string <optional>
 '' type of strand, e.g. 'ss', 'ds'. default is '' - unknown
_residue_type string <optional>
 '' type of each residue e.g. 'DNA', 'RNA'

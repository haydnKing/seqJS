Class: SpanOperator

seqJS. SpanOperator

new SpanOperator(location, operation)

An operator stores two things 1) a string defining an operation 2) a list of seqJS.SpanOperator objects or a single @seqJS.Span which are acted on by the operator Valid operations are: - 'complement': Acts on a single Span - 'join': Merges one or more Spans by 'join'ing them - 'order': Merges one or more Spans by 'order'ing them - '': no-op, zero or one Spans
Parameters:
Name Type Argument Description
location string | Array.<seqJS.Span> the location string to parse, or an array of Span to be contained. In the latter case, the second argument is required
operation string <optional>
 The operation to be carried out. If this is given then argument 1 must be an Array of Spans

Class: Record

seqJS. Record

new Record(seq, id, name, desc, annotations)

Stores a biological sequence record. Records are typically created synchronously using seqJS.read or asynchronously using seqJS.getParser. They can be written to strings using seqJS.write.
Parameters:
Name Type Description
seq seqJS.Seq the sequence object
id string the record's id
name string the record's name
desc string the record's description
annotations Object the record's annotations in {key: value, ...} form

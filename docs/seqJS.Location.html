Class: Location

seqJS. Location

new Location(_location, _operator, _location2)

Represent a single location as either and exact base (''), before a specific base ('<'), after a specific base ('>') or between two specific bases ('A.B')
Parameters:
Name Type Argument Description
_location string | number Either a string defining a location to be parsed or the position of the location
_operator string <optional>
 The operator which applies, either '', '<', '.', or '>'. If this parameter is given, then _location must be an integer. If _operator is given as '.', then _location2 is required
_location2 number <optional>
 The second location, used only when representing locations which are between two points, e.g 100.200
